🎓 Student Management System

📌 About the Project

The Student Management System is a Python console application that helps manage basic student information.

It allows users to add student details, calculate total marks and percentage, assign grades automatically, and display student records through a simple menu-driven interface.

This project was created to practice Python programming concepts and build a real-world beginner application.

---

🚀 Features

✅ Add Student

✅ Store Student Name

✅ Store Roll Number

✅ Enter Marks for 5 Subjects

✅ Automatic Total Marks Calculation

✅ Automatic Percentage Calculation

✅ Automatic Grade Calculation

✅ Display Student Details

✅ Menu-Driven Interface

✅ Exit Option
